# 📚 Bae-Code-Adventures: 알고리즘 문제 목록

하나씩 차근차근 풀어보는 알고리즘 문제 풀이 기록입니다.

## 📁 목차

### 📁 algo
- [q-2-1.py](algo/q-2-1.py) - 소문자 알파벳으로 이루어진 문자열 s가 주어진다. 이 문자열에서 ‘a’와 ‘b’의 개수가 같은 가장 긴 연속된 부분 문자열의 길이를 구해라. 반드시 연속된 구간이어야 함. 같은 개수가 되는 가장 긴 길이를 구해야 함.
- [q-2-2.py](algo/q-2-2.py) - 양의 정수로 이루어진 수열 arr가 주어진다. 이 수열의 부분 수열을 이용해 만들 수 없는 가장 작은 양의 정수를 구하라.
- [q-2.py](algo/q-2.py) - 길이가 N인 문자열이 주어집니다. 문자열에서 단어 "programmer"를 두 번 포함하는 구간을 찾으세요. 첫 번째 "programmer"는 왼쪽부터 탐색, 두 번째 "programmer"는 오른쪽부터 탐색, 두 "programmer" 사이에 존재하는 문자 개수를 구하세요. 단, 중복 사용은 불가능합니다.
- [q1-1.py](algo/q1-1.py) - 괄호 문자열이 주어질 때, 가장 깊게 중첩된 괄호의 깊이를 계산하세요.
- [q1-2.py](algo/q1-2.py) - 괄호 문자열이 주어질 때, 올바른 괄호 문자열로 만들기 위해 추가해야 하는 최소 괄호의 개수를 구하세요.
- [q1.py](algo/q1.py) - 주어진 문자열 리스트에서 각 문자열이 올바른 괄호 열림/닫힘 순서를 따르는지 판별하는 문제입니다. {}, [], () 괄호들이 짝을 맞춰 올바르게 닫혀야 함. 중간에 잘못된 순서가 나오면 "NO". 모든 괄호가 올바르게 닫히면 "YES".

### 📁 codility
- [3-2.py](algo/codility/3-2.py) - 배열 A에는 1부터 (N+1)까지의 숫자 중 하나가 빠져 있습니다. 목표: 빠진 그 하나의 숫자를 찾는 것입니다.
- [3-3.py](algo/codility/3-3.py) - 배열 A를 어느 지점 P에서 두 동강 냈을 때, |왼쪽 합 - 오른쪽 합|의 최솟값을 찾아라.
- [4-1.py](algo/codility/4-1.py) - 개구리가 강 건너편(위치 X+1)으로 가고 싶습니다. 위치 1부터 X까지 모든 위치에 나뭇잎이 떨어져야만 건널 수 있습니다. 배열 A (인덱스는 시간 K, 값 A[K]는 나뭇잎이 떨어진 위치)가 주어졌을 때, 1부터 X까지 나뭇잎이 모두 채워지는 가장 빠른 시간(인덱스)을 구하세요. (못 건너면 -1)
- [4-2.py](algo/codility/4-2.py) - 배열 A가 1부터 N까지 숫자가 하나씩 빠짐없이 다 들어있는지 확인하는 문제입니다.
- [4-3.py](algo/codility/4-3.py) - 배열 A에 없는 가장 작은 양의 정수(1, 2, 3...)를 찾으세요.
- [4-4.py](algo/codility/4-4.py) - 설명 없음 (Codility 4-4)

### 📁 DFS
- [q3-1.py](algo/DFS/q3-1.py) - 2차원 배열 grid에 1은 땅, 0은 바다로 표시됩니다. 상하좌우로 연결된 1들을 한 개의 섬으로 보고, 섬의 총 개수를 구하세요.
- [q3-2.py](algo/DFS/q3-2.py) - 0은 길, 1은 벽. (0,0)에서 시작해서 (n-1, m-1)로 도달 가능한지 DFS로 판별하세요. 도달할 수 있다면 True, 아니면 False를 출력.
- [q3-3.py](algo/DFS/q3-3.py) - 정점의 개수 n, 간선의 개수 m. m개의 간선 정보가 주어졌을 때, 연결 요소의 개수를 구하시오. 연결 요소란? DFS로 연결 가능한 노드들의 집합.
- [q3.py](algo/DFS/q3.py) - 0은 빈 칸, 1은 벽으로 구성된 2차원 배열이 주어진다. 연결된 0 덩어리(상하좌우만 연결 가능)가 몇 개인지 구해라.

### 📁 hanghae
- [day-1(bns).py](algo/hanghae/day-1(bns).py) - 설명 없음 (hanghae day-1(bns))
- [day-1.py](algo/hanghae/day-1.py) - M이상 N이하의 소수를 모두 출력하는 프로그램을 작성하시오.
- [day-11.py](algo/hanghae/day-11.py) - 설명 없음 (hanghae day-11)
- [day-12.py](algo/hanghae/day-12.py) - 효주는 포도주 시식회에 갔다. 그 곳에 갔더니, 테이블 위에 다양한 포도주가 들어있는 포도주 잔이 일렬로 놓여 있었다. 효주는 포도주 시식을 하려고 하는데, 여기에는 다음과 같은 두 가지 규칙이 있다. 포도주 잔을 선택하면 그 잔에 들어있는 포도주는 모두 마셔야 하고, 마신 후에는 원래 위치에 다시 놓아야 한다. 연속으로 놓여 있는 3잔을 모두 마실 수는 없다. 효주는 될 수 있는 대로 많은 양의 포도주를 맛보기 위해서 어떤 포도주 잔을 선택해야 할지 고민하고 있다.
- [day-13.py](algo/hanghae/day-13.py) - JadenCase란 모든 단어의 첫 문자가 대문자이고, 그 외의 알파벳은 소문자인 문자열입니다. 단, 첫 문자가 알파벳이 아닐 때에는 이어지는 알파벳은 소문자로 쓰면 됩니다. 문자열 s가 주어졌을 때, s를 JadenCase로 바꾼 문자열을 리턴하는 함수, solution을 완성해주세요.
- [day-14.py](algo/hanghae/day-14.py) - N 초 동안 싸움을 함. 플레이하는 캐릭은 A,B 를 사용할수있다 스킬. a는 1초 b는 m초의 시전시간이다. n이 4초이고 m이 2초이면 aaaa,aab,aba,bb로 5가지가 가능. 가능한 조합의 수를 1,000,000,007로 나눈 나머지 값을 출력한다.
- [day-2.py](algo/hanghae/day-2.py) - 자연수 n을 입력받아 n번째 피보나치 비스무리한 수열을 구해보자! 피보나치 비스무리한 수열은 f(n) = f(n-1) + f(n-3)인 수열이다. f(1) = f(2) = f(3) = 1이다.
- [day-3.py](algo/hanghae/day-3.py) - 컴퓨터 바탕화면 상태를 나타낸 문자열 배열 wallpapaer가 주어집니다. 파일들은 바탕화면의 격자칸에 위치하고, 빈칸은 ".", 파일이 있는 칸은 "#"입니다. 최소한의 이동거리를 갖는 한 번의 드래그로 모든 파일을 선택해서 한 번에 지우려고 할 때, 드래그의 시작점 (lux, luy)과 끝점 (rdx, rdy)을 담은 정수 배열 [lux, luy, rdx, rdy]를 return하면 됩니다.
- [day-4.py](algo/hanghae/day-4.py) - 행과 열의 크기 N, 높이정보는 N * N 2차원 배열 형태로 주어집니다. 각 원소는 지점 높이를 표시합니다. 장마철에 물에 잠기지 않는 안전한 영역의 최대 개수를 계산하는 프로그램을 작성하시오.
- [day-5.py](algo/hanghae/day-5.py) - 매일 아침 9시에 학교에서 측정한 온도가 어떤 정수의 수열로 주어졌을 때, 연속적인 며칠 동안의 온도의 합이 가장 큰 값을 알아보고자 한다.
- [day-7.py](algo/hanghae/day-7.py) - 쇠막대기는 자신보다 긴 쇠막대기 위에만 놓일 수 있다. 쇠막대기를 다른 쇠막대기 위에 놓는 경우 완전히 포함되도록 놓되, 끝점은 겹치지 않도록 놓는다. 각 쇠막대기를 자르는 레이저는 적어도 하나 존재한다. 레이저는 어떤 쇠막대기의 양 끝점과도 겹치지 않는다. 레이저는 여는 괄호와 닫는 괄호의 인접한 쌍 ‘( ) ’ 으로 표현된다. 또한, 모든 ‘( ) ’는 반드시 레이저를 표현한다. 쇠막대기의 왼쪽 끝은 여는 괄호 ‘ ( ’ 로, 오른쪽 끝은 닫는 괄호 ‘) ’ 로 표현된다. 쇠막대기와 레이저의 배치를 나타내는 괄호 표현이 공백없이 주어졌을 때, 잘려진 쇠막대기의 총 개수를 구하세요.
- [day-8.py](algo/hanghae/day-8.py) - 파일의 개수 N이 주어지고, 패턴이 주어진다. 패턴은 알파벳 소문자와 별표 한 개로 이루어져 있으며, 별표는 문자열의 시작과 끝에 있지 않다. 다음 N개 줄에는 파일 이름이 주어진다. 각 파일 이름이 패턴과 일치하면 "DA", 일치하지 않으면 "NE"를 출력한다.
- [day-9.py](algo/hanghae/day-9.py) - N개의 저울추가 주어졌을 때, 이 추들로 측정할 수 없는 양의 정수 무게 중 최솟값을 출력한다.

### 📁 sliding_window
- [q1.py](algo/sliding_window/q1.py) - DNA 문자열 길이 |S|, 부분문자열의 길이 |P|. DNA 문자열이 주어지고, 부분문자열에 포함되어야 할 {‘A’, ‘C’, ‘G’, ‘T’} 의 최소 개수가 공백을 구분으로 주어질 때, 조건을 만족하는 부분 문자열의 개수를 구하는 문제입니다.
